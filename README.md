# Triquest Remake 

A dot intenship challenge test project made using React and Typescript that follows the [Bulletproof React Architecture](https://github.com/alan2207/bulletproof-react).

## Tech Stack
- React
- Typescript
- Headless UI
- Tailwind
- React Query
- Axios
